# CougarPetroleum
Welcome to the CougarPetroleum!
This website is designed to provide you with a comprehensive and user-friendly platform for gas modeling and price prediction. 


The project is a web-based task management application that follows the Model-View-Controller (MVC) architectural pattern. It allows users to create, organize, and track their gas usage in a structured and efficient manner.
This project demonstrates how the MVC architectural pattern can be effectively utilized to create a well-organized, scalable, and user-friendly task management application.
